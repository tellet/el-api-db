# Common queries

A collection of common SQL queries.

## Prerequisites
* Docker

## Run via Docker Compose

```shell
git clone https://github.com/tellet/el-api-db.git
cd el-api-db
docker compose up --build -d
```

The app will be accessible by [http://127.0.0.1](http://127.0.0.1)

## Endpoints

* /products
* /categories
* /pairs
